# DESIGN NOTES

## Image Placement
All 4 images should be used throughout:
1. `hero-luxury-room.jpg` - Cover page background
2. `hotel-staff-management.jpg` - Page 3 (Solution)
3. `kitchen-operations.jpg` - Page 4 (Results)
4. `room-service-cart.jpg` - Page 5 (Technology)

## Consistency
- Maintain 20mm margins on all pages
- Use copper (#C87533) for all CTAs and highlights
- Keep backgrounds dark (charcoal) for luxury feel
- White text for maximum contrast
- Generous whitespace - don't overcrowd

## Typography Scale
- H1 (Main Headlines): 42-48pt Outfit Bold
- H2 (Section Headers): 32-38pt Outfit Bold
- H3 (Subsections): 24-28pt Outfit Medium
- Body: 14-16pt Inter Regular
- Small text: 12pt Inter Regular

## Icons
Use simple, modern icons in copper color:
- 💰 Dollar sign for revenue
- ⚡ Lightning for speed/efficiency
- ⭐ Star for luxury/quality
- 📊 Chart for metrics
- 🔒 Lock for security
- ✓ Checkmark for benefits

## Print Specifications
- Size: A4 (210mm × 297mm)
- Color Mode: CMYK
- Resolution: 300 DPI minimum
- Bleed: 3mm on all sides
- File format: PDF/X-1a:2001
